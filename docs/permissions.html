Permissions

Claude Code has a flexible permission system for tool execution.

Permission Modes

Default (ask-first)

Claude asks before executing potentially dangerous tools like Bash, Edit, Write.

Auto Mode

Automatically approves safe tools. Dangerous tools still require confirmation.

YOLO Modes

Progressive bypass levels:

  • yoloLite - YOLO Lite: Auto-allow most actions with guardian checks
  • yolo - YOLO ALLOW: Broad auto-approval for trusted sessions
  • yoloMax - YOLO MAX: Full autonomy and sandbox bypass when available

Switching Modes

/yolo          # Open YOLO mode picker
/yolo yolo     # Directly switch to yolo
/yolo default  # Back to ask-first

Configuration

{
  "permissions": {
    "mode": "auto",
    "autoApproveReadOnly": true
  }
}
Warning: YOLO modes can bypass safety checks. Use with caution!
