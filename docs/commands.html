Commands

Claude Code provides 90+ slash commands for common development tasks. Type / to see all available commands.

CommandDescription

/editEdit specific part of file
/globFind files by pattern
/grepSearch file contents
/add-dirAdd directory to project
/pathShow file path
/renameRename files
