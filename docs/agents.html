Agents

Claude Code supports subagents for parallel task execution. Create AI agents that work independently in the background.

Using Agents

Create an Agent

/agent Create a new component for the dashboard

List Agents

/agents

Resume Agent

/resume

Built-in Agents

AgentDescription
general-purposeGeneral coding tasks
exploreExplore codebase and answer questions
planPlan and structure complex tasks
verificationVerify code correctness
claude-code-guideHelp learn Claude Code

Agent Features

  • Background Execution - Agents run in background while you continue
  • Parallel Agents - Run multiple agents simultaneously
  • Memory - Agents can maintain context across tasks
  • Worktree Support - Agents can use git worktrees

Agent Commands

/agent [prompt]          # Create agent with prompt
/agents list              # List running agents
/agent kill [id]          # Kill an agent
/resume                   # Resume previous agent
Tip: Use agents for parallel file operations or complex tasks that can run independently.
