Tool Permissions

Tools are categorized by risk level:

  • Read-only: Read, Glob, Grep - Usually auto-approved
  • Non-destructive: Write, Edit, Bash (safe) - May require confirmation
  • Destructive: rm, git push --force - Always require confirmation
Tip: Use /yolo to bypass tool permissions for automation.
