Custom Keybindings

Create ~/.claude/keybindings.json to override default bindings:

[
  {
    "context": "Chat",
    "bindings": {
      "ctrl+enter": "chat:submit",
      "meta+k": "chat:clear"
    }
  },
  {
    "context": "Global",
    "bindings": {
      "ctrl+shift+r": "app:reload"
    }
  }
]

Validation

The keybinding system validates user bindings against reserved shortcuts. The following keys cannot be rebound:

  • Ctrl+C — Reserved for interrupt
  • Ctrl+D — Reserved for exit

Attempting to rebind reserved keys produces a validation error.

Platform Notes:
  • Windows: Cmd/Super modifier keybindings are supported via the Kitty keyboard protocol. Image paste uses Alt+V (since Ctrl+V is system paste).
  • macOS: Meta maps to Cmd. Use Ctrl+Shift+T as fallback if Option is not bound as Meta.
  • Kitty Protocol: Cmd+* and Super+* bindings only fire on terminals supporting the Kitty keyboard protocol (Kitty, WezTerm, Ghostty, iTerm2).
Tip: Use the /keybindings command or the bundled keybindings skill to view and customize your current binding configuration interactively.
