Environment Variables

Complete reference for environment variables that control Clew.

Debug & Logging

VariableEffect
DEBUG=1Enable all debug logging
DEBUG=provider:anthropicDebug specific provider
DEBUG=provider:*Debug all providers
DEBUG=toolsDebug tool execution
DEBUG=bridgeDebug bridge/WebSocket connections
VERBOSE=1Enable verbose output

Terminal & Display

VariableEffect
NO_COLOR=1Disable all colors in output
FORCE_COLOR=1Force color output (overrides NO_COLOR)
CLICOLOR=0Disable colors (standard convention)
TERM=xterm-256colorTerminal type detection

Runtime

VariableEffect
NODE_OPTIONSNode/Bun runtime options
BUN_RUNTIME=1Set automatically when running under Bun
CLEW_HOMEOverride the home directory for .claude/ config
CLEW_CONFIG_DIROverride the config directory path

Usage Example

# Set in shell
export OPENAI_API_KEY=sk-...
export DEBUG=1
clew

# Set inline for one session
DEBUG=provider:deepseek clew

# Using .env file (auto-loaded from project root)
echo "OPENAI_API_KEY=sk-..." >> .env
echo "DEEPSEEK_API_KEY=sk-..." >> .env
Note The providers.json file in src/services/ai/providers.json contains the authoritative list of all supported env vars and their provider mappings.
