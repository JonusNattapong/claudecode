Troubleshooting

Common Issues

Build Failures

Windows CRLF Bug: On Windows, Edit tool converts files to CRLF. Use Python to fix: 
python -c "import sys; data=open('file.ts','rb').read(); open('file.ts','wb').write(data.replace(b'\r',b''))"

TypeScript Errors

bun x tsc --noEmit

Note: Pre-existing template literal errors in kanban.ts exist but tests still pass.

API Key Issues

  • Check environment variables are set
  • Verify API key format is correct
  • Ensure you have credits/quota on the provider

Provider Not Working

/model list   # See available models

Debug Mode

DEBUG=1 bun run src/main.tsx session
DEBUG=provider:anthropic bun run src/main.tsx session

In-Session Diagnostics

/status    # Show internal state
/doctor    # Run diagnostics & auto-fix
/context   # View context window usage

Getting Help

  • Check GitHub Issues
  • Review debug logs with DEBUG=1
  • Use /doctor command
