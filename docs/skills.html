Skills

Skills are specialized instruction sets that guide Claude Code for specific tasks.

Using Skills

Load a Skill

/skill skill-name

List Available Skills

/skills

Built-in Skills

SkillDescription
verifyVerification workflows - test, lint, typecheck
debugDebugging assistance - logs, stack traces, breakpoints
web-searchWeb search optimization
update-configConfiguration management
rememberRemember user preferences
simplifySimplify complex code
batchBatch file operations

lorem-ipsumGenerate placeholder content

Plugin Skills

Plugins can add their own skills. Check plugin documentation for details.

Creating Skills

Skills are defined in .claude-plugin/skills/ folder of plugins.

Tip: Use skills to optimize Claude's behavior for specific workflows.
