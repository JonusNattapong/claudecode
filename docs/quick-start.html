Quick Start

Get up and running with Claude Code in 5 minutes.

Step 1: Install

git clone https://github.com/JonusNattapong/ClaudeCode.git
cd claude-code
bun install

Step 2: Configure API Key

# Option A: Environment variable
export ANTHROPIC_API_KEY=sk-ant-...

# Option B: .env file
echo "ANTHROPIC_API_KEY=sk-ant-..." > .env

Step 3: Run

# Development mode
bun run dev

# Or build and run
bun run build
bun run src/main.tsx session

Step 4: Start Coding!

Basic Commands

/read src/main.tsx       # Read a file
/edit                       # Edit a file
/write                      # Create a new file
/glob **/*.ts               # Find files
/grep function foo          # Search content
/git status                 # Git operations
/test                       # Run tests

AI Commands

/model claude-sonnet-4-20250514  # Switch model
/model openai/gpt-4o           # Switch provider
/provider openai               # Change provider
/model list                    # See available models

Permissions

/yolo         # Open YOLO mode picker
/yolo yolo    # Auto-approve tools
/yolo god     # Full autonomy
