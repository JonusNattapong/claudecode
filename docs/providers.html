AI Providers

Claude Code supports 11 AI providers with a unified interface. Switch between providers seamlessly.

Supported Providers

1. Anthropic (Default)

ANTHROPIC_API_KEY=sk-ant-...

Models: claude-sonnet-4-20250514, claude-opus-4-5-20250514, claude-3-5-sonnet-20241022, claude-3-haiku-20240307

The default provider with best tool-use capabilities.

2. OpenAI

OPENAI_API_KEY=sk-...

Models: gpt-4o, gpt-4o-mini, gpt-4-turbo, gpt-3.5-turbo

OpenAI's latest models including GPT-4o.

3. Google Gemini

GOOGLE_GENERATIVE_API_KEY=...

Models: gemini-2.0-flash-exp, gemini-1.5-pro, gemini-1.5-flash

Google's Gemini models with strong vision capabilities.

4. OpenRouter

OPENROUTER_API_KEY=sk-or-...

Models: 100+ models including anthropic/claude-*, openai/gpt-*, google/gemini-*, meta/llama-*, mistral/*

Access to 100+ models through unified API.

5. Ollama (Local)

OLLAMA_HOST=http://localhost:11434

Models: llama3, llama3.1, mistral, codellama, phi3, qwen2

Run models locally on your machine. No API costs.

6. xAI Grok

XAI_API_KEY=...

Models: grok-2, grok-2-vision

Elon's xAI Grok models.

7. Mistral

MISTRAL_API_KEY=...

Models: mistral-large-latest, codestral-latest

Mistral's large language models.

8. OpenAI Compatible

OPENAI_COMPATIBLE_API_KEY=...
OPENAI_COMPATIBLE_BASE_URL=https://your-api.com/v1

Connect to any OpenAI-compatible API (LM Studio, local APIs, etc.)

9. ChatGPT Session (OAuth)

CHATGPT_SESSION_TOKEN=...

Use your ChatGPT Plus/Pro subscription via browser session token.

10. Copilot

GITHUB_TOKEN=ghp_...

Use GitHub Copilot for code completion.

11. KiloCode

KILOCODE_API_KEY=...

KiloCode provider for specialized tasks.

Switching Providers

Using /provider command

/provider anthropic
/provider openai
/provider google

Using /model command

/model claude-sonnet-4-20250514
/model openai/gpt-4o
/model google/gemini-2.0-flash-exp

Via Configuration

{
  "provider": "openai",
  "model": "gpt-4o"
}

Model Discovery

/model list           # List all available models
/model list anthropic # List Anthropic models only
Note: Not all providers support all features. Some models may not support tools/messages.
