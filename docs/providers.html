AI Providers

Claude Code supports 15 AI providers through a unified interface. Switch between providers and models at runtime without restarting.

Supported Providers

1. Anthropic (Default)

ANTHROPIC_API_KEY=sk-ant-...

Models: claude-opus-4-7, claude-sonnet-4-7, claude-opus-4-6, claude-haiku-4-5, claude-sonnet-4-5, and earlier Claude models

2. OpenAI

OPENAI_API_KEY=sk-...

Models: gpt-5.5-pro, gpt-5.5, gpt-5.4-mini, gpt-4o, o1, o3, and others

3. Google Gemini

GOOGLE_API_KEY=...

Models: gemini-3.1-pro, gemini-3.1-flash, and earlier Gemini models

4. OpenRouter

OPENROUTER_API_KEY=sk-or-...

Models: 100+ models from Anthropic, OpenAI, Google, Meta, Mistral, xAI, DeepSeek, and more — all routed through a single API

5. DeepSeek

DEEPSEEK_API_KEY=...

Models: deepseek-v4-pro, deepseek-v4-flash

6. Ollama (Local)

OLLAMA_HOST=http://localhost:11434

Models: llama4:70b, deepseek-v4, llama3.3, and any model pulled in Ollama

7. xAI Grok

XAI_API_KEY=...

Models: grok-4-20, grok-4-3, grok-4

8. Mistral

MISTRAL_API_KEY=...

Models: mistral-large-4, mistral-small-latest, mistral-large-latest, ministral-3-14b

9. Groq

GROQ_API_KEY=...

Models: llama-4-70b, llama-4-8b — high-speed inference via Groq LPU

10. Copilot

COPILOT_GITHUB_TOKEN=ghp_...

Use GitHub Copilot for AI-assisted code generation via your GitHub account.

11. KiloCode

KILOCODE_API_KEY=...

Multi-provider access through KiloCode with OpenAI, Anthropic, DeepSeek, Kimi, and MiMo models.

12. OpenCode

OPENCODE_API_KEY=...

Models: claude-opus-4-7, claude-sonnet-4-6, gpt-5.5, gemini-3.1-pro, kimi-k2.6, glm-5.1, minimax-m2.7

13. Cline API

CLINE_API_KEY=...

Models: claude-opus-4-7, claude-sonnet-4-6, gpt-5.5, gemini-3.1-pro, minimax-m2.5

14. ChatGPT Plus (OAuth)

CHATGPT_SESSION_TOKEN=...

Leverage your ChatGPT Plus subscription via browser session token authentication.

Switching Providers

Using /provider Command

/provider anthropic
/provider openai
/provider google

Using /model Command

/model claude-sonnet-4-20250514
/model openai/gpt-4o
/model google/gemini-2.5-pro

Via Configuration File

{
  "provider": "openai",
  "model": "gpt-4o"
}

Model Discovery

/model list           # List all available models
/model list anthropic # List Anthropic models only
/model list openai    # List OpenAI models only
Model Caching: Available models are cached for 5 minutes to avoid excessive API calls. Use /model refresh to force a refresh.
Feature Availability: Not all providers support every feature. Tool calling, streaming, vision, and system prompts vary by provider and model. Claude models via Anthropic API provide the most comprehensive tool-use experience.
