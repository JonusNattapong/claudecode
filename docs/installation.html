Installation

Prerequisites

  • Bun — JavaScript runtime (bun.sh)
  • Git — For version control
Note: Claude Code is built and tested with Bun. Node.js may work but is not fully tested.

Install Steps

1. Clone Repository

git clone https://github.com/anomalyco/claude-code.git
cd claude-code

2. Install Dependencies

bun install

3. Build

bun run build

4. Run

# Development mode (with hot reload)
bun run dev

# Or production build
bun run src/main.tsx session

API Key Setup

Option 1: Environment Variable

# Linux/Mac
export ANTHROPIC_API_KEY=sk-ant-...

# Windows PowerShell
$env:ANTHROPIC_API_KEY="sk-ant-..."

Option 2: .env File

# Create .env file in project root
echo "ANTHROPIC_API_KEY=sk-ant-..." > .env

Option 3: Multiple Providers

ANTHROPIC_API_KEY=sk-ant-...
OPENAI_API_KEY=sk-...
GOOGLE_GENERATIVE_API_KEY=...

Verifying Installation

# Run dev mode
bun run dev

# Inside session, test commands
/model list
/stats
/help
