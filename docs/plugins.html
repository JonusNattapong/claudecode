Plugins

Extend Claude Code functionality with plugins. Plugins can add commands, agents, skills, and hooks.

Using Plugins

Install a Plugin

/plugin install plugin-name

Reload Plugins

/reload-plugins

List Installed Plugins

/plugins

Available Plugins

plugin-dev

Development toolkit for creating Claude Code plugins.

  • Create new plugins
  • Add commands, skills, agents
  • Plugin structure guides
  • Skill development
/plugin dev create my-plugin

feature-dev

Feature development workflow with AI agents.

  • Code reviewer agent
  • Code architect agent
  • Code explorer agent
  • Feature development flow

code-review

Automated code review tools.

  • PR code review
  • Code quality analysis
  • Best practices checker

hookify

Git hooks integration for better workflows.

  • Pre-commit hooks
  • Custom hook rules
  • Security scanning
/hookify enable
/hookify add "*.sh" --require-tests

commit-commands

Enhanced git commit experience.

  • Smart commit messages
  • Auto-staging
  • Conventional commits

frontend-design

UI/UX design assistance.

  • Tailwind guidance
  • Component suggestions
  • Design system help

pr-review-toolkit

Pull request review automation.

  • PR test analyzer
  • Type design analyzer
  • Comment analyzer
  • Silent failure hunter

Agent SDK development tools.

  • SDK app templates
  • TypeScript/Python verification
  • SDK integration help

Skills are specialized instructions for specific tasks:

SkillDescription
/skill verifyVerification workflows

/skill update-configConfig management
/skill rememberRemember user preferences
/skill simplifySimplify code
/skill batchBatch file operations

Creating Plugins

/plugin dev create my-plugin

Plugin structure:

my-plugin/
├── .claude-plugin/
│   ├── plugin.json     # Manifest
│   ├── skills/        # Skill implementations
│   └── hooks/          # Hook handlers
├── commands/           # Custom commands
├── agents/            # Custom agents
├── skills/            # Skills
└── README.md
Info: See plugins/ directory for plugin examples.
