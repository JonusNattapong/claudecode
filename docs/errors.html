Error Reference

Common errors, their causes, and solutions.

API & Provider Errors

ErrorCauseFix
No API key found for provider Missing environment variable for the selected provider Set the appropriate API key (e.g. export OPENAI_API_KEY=sk-...)
401 Unauthorized Invalid or expired API key Verify your API key is correct and hasn't expired
429 Rate Limited Too many requests Wait and retry, or switch to a different provider/model
Model not found The model name doesn't exist or isn't available for your API key Run /model list to see available models
Context length exceeded Conversation exceeds model's context window Run /compact or /clear to free context

Installation & Launch Errors

ErrorCauseFix
command not found: clew Binary not in PATH Verify npm global bin is in PATH, or install with bun install -g
Bun not found clew requires Bun at runtime Install Bun from bun.sh
Export named '...' not found Running from source with missing exports Run bun run build first, or check for compile errors
Cannot find module Missing dependencies Run bun install to install dependencies

Configuration Errors

ErrorCauseFix
Invalid settings.json JSON syntax error in settings file Validate the file with bun run jsonlint or check for trailing commas
Hook command failed A hook shell command exited with non-zero code Run the hook command manually to debug. Check /doctor for hook status
Plugin load failed Plugin manifest invalid or missing dependencies Run /plugin list to see which plugins failed. Check plugin logs
MCP server connection failed MCP server command or URL is unreachable Verify the server is running. Check /mcp list for connection status

Runtime Errors

ErrorCauseFix
Permission denied Tool execution blocked by permission rules Press Shift+Tab to cycle permission modes, or use /permissions
Tool execution timeout A tool took too long to complete Check if the command is hung. Increase timeout in tool config if needed
Bridge connection lost WebSocket disconnected Run /bridge reconnect to reconnect
Session not found Trying to resume a session that doesn't exist Use /resume to see the list of available sessions

Diagnostics

Run /doctor at any time to run automated diagnostics:

> /doctor
✓ Environment check passed
✓ API key configured (OpenAI)
✓ Settings file valid
✓ Hooks: 3 configured, all OK
✓ Plugin: 5 loaded, 0 errors
! MCP: 1 server unreachable
→ Run /mcp list for details

For more detailed debugging, launch with clew --debug or set DEBUG=1.

Report an Issue

If you encounter a bug not listed here, please report it on GitHub Issues with:

  • Clew version (clew --version)
  • OS and terminal emulator
  • Steps to reproduce
  • Full error output (run with --debug)
