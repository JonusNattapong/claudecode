Claude Code

A research-focused fork of Anthropic's Claude Code CLI with multi-provider routing, extensible plugin architecture, and enhanced AI capabilities.

🚀 Get Started

Start using Claude Code in minutes:

# Install dependencies
bun install

# Build the project
bun run build

# Start a session
bun run src/main.tsx session

✨ What's New (2026)

  • 11 AI Providers — Anthropic, OpenAI, Google Gemini, OpenRouter, Ollama, xAI, Mistral, and more
  • 90+ Commands — File ops, git, search, npm, agents, MCP, and more
