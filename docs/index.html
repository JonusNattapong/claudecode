Claude Code

A research fork of Anthropic's Claude Code CLI — a terminal-based AI coding assistant with multi-provider routing, extensible plugin architecture, and a comprehensive tool system for professional development workflows.

Get Started

Install and run Claude Code in minutes:

# Clone the repository
git clone https://github.com/JonusNattapong/ClaudeCode.git
cd claudecode

# Install dependencies
bun install

# Build and start a session
bun run build
bun run src/main.tsx session

Key Capabilities

  • 15 AI Providers — Anthropic, OpenAI, Google Gemini, OpenRouter, Ollama, DeepSeek, xAI, Mistral, Groq, Copilot, KiloCode, OpenCode, OpenCode Go, Cline, ChatGPT Plus
  • 104 Slash Commands — File operations, git, search, AI model management, MCP servers, plugins, and diagnostics
  • 56 Built-in Tools — Read, Edit, Write, Glob, Grep, Bash, PowerShell, WebFetch, WebSearch, Agent, Task, MCP, LSP, Git, CodeIndex, and more
  • 13 Plugins — Agent SDK, code review, commit commands, frontend design, feature development, plugin development, and more
  • 20 Bundled Skills — Debugging, web search, web scraping, commit, code review, scheduling, keybinding configuration
  • Permission System — Four-tier automation mode: Default approval, YOLO Lite, YOLO ALLOW, and YOLO MAX
  • Agent View — Dispatch and manage many sessions from one screen with supervisor daemon, IPC, worktree isolation, and AI row summaries
  • Smart Memory — Cross-lingual semantic search with embeddings (100+ languages), hybrid recall (local + LLM), auto-memory, and team memory sync
  • Bridge Mode — WebSocket-based remote collaboration with session sharing and team onboarding

Recent Updates (v2.1.140)

  • Smart Memory with Semantic Search — Cross-lingual memory recall using IBM Granite 97M multilingual embeddings (100+ languages), hybrid search (local + LLM fallback), embedding cache, and /memory-search command
  • Subagent API Headers (G32) — Subagent API requests now carry x-claude-code-agent-id and x-claude-code-parent-agent-id headers for traceability
  • API Key Override Gates (G33) — OAuth-dependent features are disabled when API key overrides are active, with clear messaging
  • Compaction Preserves Instructions (G26) — Context compaction now preserves sensitive user instructions verbatim
  • Fallback Thinking Toggle (E61) — Added Ctrl+Shift+T as fallback hotkey for toggling thinking mode on macOS
  • Symlinked Settings Hot-Reload (G8) — Settings file watcher resolves realpath, enabling hot-reload for symlinked settings.json
  • Sandbox Auto-Allow Fix (G9) — Shell expansion commands ($VAR, $(cmd)) now respect auto-allow before prompting
  • SIGTSTP Handler (E50) — Ctrl+Z no longer hangs in npx/bun wrapper environments
  • Diff Truncation Fix (G41) — Corrected off-by-one in line count when truncation boundary falls on a newline
  • Regex Metachar Fix (G44) — Special regex characters in skill argument names are now escaped before substitution
  • Keybinding Modifier Fix (G19) — Cmd/Super/Win keybindings no longer incorrectly flagged as unparseable
