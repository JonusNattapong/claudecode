Frequently Asked Questions

General

What is Claude Code?

A research-focused fork of Anthropic's Claude Code CLI with multi-provider routing, plugin architecture, and enhanced features.

How is this different from official Claude Code?

This fork adds: multiple AI providers, plugin system, enhanced YOLO modes, and custom features.

Is this legal?

See LEGAL.md for details. This is research-focused code from an npm disclosure.

Installation

Why Bun? Can I use Node.js?

Bun is the primary runtime. Node.js may work but is not fully tested.

API keys not working?

Check environment variables are set correctly. See Providers page.

Usage

How do I switch models?

/model claude-sonnet-4-20250514
/model gpt-4o

How do I use plugins?

/plugin install plugin-name

What are YOLO modes?

Permission bypass modes for automated tool execution. See Permissions.

Troubleshooting

Build errors?

bun run build
# If issues persist:
rm -rf node_modules
bun install

TypeScript errors?

bun x tsc --noEmit
